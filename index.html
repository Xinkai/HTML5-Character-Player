A & Q

Which kind of file can be played?

The video files on your hard drive with the right format. The format depends on which browser you're using.

Browser Mp4 WebM Ogg
Internet Explorer yes no no
Chrome yes yes yes
Firefox no yes yes
Safari yes no no
Opera no yes yes

Note:It is possible that you have a file that ends with ".mp4" but cannot be played with Chrome. Extension names doesn't tell everything about a file.
Bad Apple has been tested again on Google Chrome. You can download Bad Apple.

File not opening / No sound

It might be a compatibility issue.
Using the latest Google Chrome on a not-too-old computer is recommended.

Extremely slow / laggy

Due to the technical reasons, this software can be resource hogging.
Do not even try running this on a mobile device or an old computer.
You can try increasing pixelate size, by which the CPU workload decreases.
